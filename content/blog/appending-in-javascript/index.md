Create text node on document
```javascript
Document.createTextNode(text);
```
Append node on Node
```javascript
Node.appendChild(node);
```
Remove node on Node
```javascript
Node.removeChild(node);
```
Collection of child nodes on Node
```javascript
Node.childNodes;
```
Length of child nodes collection
```javascript
Node.childNodes.length;
```
